# 強化内容

## ステータス強化系

### ほぼ全ての敵

攻撃力、体力が強化されています。
体力が自然回復します。

### ゾンビ系

ノックバックしにくくなっています。


### ファントム

ファントム召喚トロッコスポナーを背負ってスポーンします。
ノックバックが強化されています。


# 追加ボス系モンスター

ごく低確率で、敵が強力なボスモンスターに変化します。
倒すと強力な装備が手に入るかもしれません。
