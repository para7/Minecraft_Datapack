# ウィザースケルトン発光データパック

ウィザースケルトンの頭を集めるの、大変ですよね？
かと言って、いきなりネザー要塞トラップを組むのもつらい…
そんなあなたにウィザースケルトン狩りを助けるこのデータパックです。
バニラで気兼ねなく使えるゲームバランスになるよう配慮しています。


# 作り方

材料をエンチャントテーブルの上に投げてください。

・黒骸の杖レプリカ
人参付きの棒x1
石炭ブロックx1

右クリックで経験値を3消費し、近くのウィザースケルトンを短時間発光させます。


・黒骸の杖
人参付きの棒x1
ウィザースケルトンの頭x1

右クリックで経験値を3消費し、広範囲のウィザースケルトンを長時間発光させます。


・真・黒骸の杖
人参付きの棒x1
ネザースターx1

装備するだけで超広範囲のウィザースケルトンを発光させます。


# 使い方

杖を持って右クリックしてください。
ウィザースケルトンが発光します。
一部の杖は経験値レベルを消費します。

なお経験値が足りない場合は何も起きません。
そのうちその旨を伝えるメッセージを実装したいと思います
